# The Brittle Bunch 🍬

Official website for **The Brittle Bunch** — organic, homemade sweet treats made by kids, for everyone.

## About

The Brittle Bunch is a kid-run small business selling handcrafted organic confections, starting with their signature **Almond Brittle**. Gluten-free options available.

## Website Features

- **Home** — Hero section with brand intro and featured product
- **Shop** — Product catalog (Almond Brittle + coming-soon items)
- **Order** — Online order form with quantity selector and live total
- **Find a Store** — ZIP code-based store locator
- **Our Story** — Company overview and team bios
- **Contact** — Contact form with subject categories

## Hosting on GitHub Pages

1. Push this repository to GitHub
2. Go to your repo → **Settings** → **Pages**
3. Under **Source**, select `main` branch and `/ (root)` folder
4. Click **Save** — your site will be live at `https://yourusername.github.io/brittle-bunch/`

## Tech

- Pure HTML, CSS, and vanilla JavaScript — no build tools or dependencies needed
- Logo and assets embedded directly in the HTML (no broken image links)
- Google Fonts: Baloo 2, Playfair Display, DM Sans

## Customization

Open `index.html` in any text editor to update:
- Contact email and phone number (search `thebrittlebunch.com` and `555`)
- Kid bios in the **Our Story** section (search `team-placeholder`)
- Real store locations (search `MOCK_STORES`)
- Pricing (search `const PRICE`)
